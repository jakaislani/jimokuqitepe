What area code is 458 belong to?

SALEM, Ore. — The Public Utility Commission has added a telephone area code for much of Oregon: 458. Beginning in 2010, it will be assigned in the area that uses 541 — all of the state outside the populous northwest corner.

What is a 458 number?

The 458 number is a North American area code that serves the northwest part of Oregon and its suburbs like Eugene, Medford, Springfield, Corvallis, and more. The 458 area code is an overlay for the 503 and 541 area codes that was used for the entire state of Oregon.
